📚 Book Store Web Application with Admin Panel

A full-stack Book Store web application where users can browse books, add them to their cart, and place orders. The system also includes an Admin Panel that allows administrators to manage books, users, and orders.

This project demonstrates full-stack development using React, Node.js, Express, and MongoDB, along with modern state management and REST APIs.

🚀 Features
👤 User Features

User registration and login

Browse available books

View book details

Add books to cart

Place orders

Responsive UI

🛠 Admin Features

Admin login

Add new books

Update book information

Delete books

Manage users

Manage orders
